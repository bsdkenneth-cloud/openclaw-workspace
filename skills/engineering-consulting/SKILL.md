# Engineering Consulting Skill

## Purpose
Use this skill for:
- engineering troubleshooting
- technical diagnosis
- repair recommendation drafting
- material selection support
- industrial problem analysis
- practical consulting summaries

## Rules
- Be technically grounded and explicit.
- Separate observation, likely cause, risk, and recommendation.
- Avoid pretending uncertainty does not exist.
- Prefer actionable engineering language.

## Output Style
- technical summary
- root cause analysis
- repair recommendation
- risk notes
