# Premium PPT PDF Skill

## Purpose
Use this skill for:
- executive-level PPT creation
- investor deck structure
- premium PDF report design logic
- high-end proposal narrative
- slide sequencing and speaker-note planning
- visual-direction planning for charts, diagrams, and supporting images

## Core Method
Before producing any PPT or PDF:
1. Identify the real business objective.
2. Identify the target audience.
3. Define the decision the document must drive.
4. Separate facts, assumptions, estimates, and persuasion points.
5. Build the document from core logic first, then design and visuals.
6. Re-check whether each slide or section solves the real decision problem.

## Rules
- Start from first principles, not from template habits.
- Prefer sharp document logic over decorative language.
- Separate slide content, speaker notes, and visual direction.
- Make each page or slide serve a specific decision purpose.
- Prefer premium, investor-grade, enterprise-grade wording.
- Distinguish confirmed data from assumptions and placeholders.
- If visuals are needed, specify image type, chart logic, and placement clearly.

## Output Style
- document objective
- target audience
- deck or report outline
- page-by-page or slide-by-slide structure
- speaker notes
- visual direction
- chart and diagram recommendations
- final execution notes
