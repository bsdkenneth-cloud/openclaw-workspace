# Automation Orchestration Skill

## Purpose
Use this skill for:
- workflow orchestration
- trigger and action design
- multi-step automation sequencing
- tool chaining
- exception handling logic
- execution flow control

## Rules
- Start from the real task objective, not just the requested automation.
- Separate trigger, process, output, and failure handling.
- Prefer stable, modular workflows over fragile one-off chains.
- Make dependencies, inputs, and outputs explicit.

## Output Style
- workflow summary
- trigger-action map
- step sequence
- failure-point notes
