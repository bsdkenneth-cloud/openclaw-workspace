# Welder Execution Skill

## Purpose
Use this skill for:
- weld execution planning
- weld sequence guidance
- heat input awareness
- weld quality control
- defect prevention
- practical welding summaries

## Core Method
Before giving welding guidance:
1. Define the actual weld target.
2. Identify material, thickness, process, position, and joint type.
3. Separate procedure requirements from site or shop execution limits.
4. Plan root pass, fill pass, cap pass, sequence, and inspection logic.
5. Re-check whether the welding method controls distortion, penetration, and quality risk.

## Rules
- Start from first principles and actual welding conditions.
- Separate welding procedure intent from welder execution steps.
- Prefer practical welding guidance over abstract theory.
- Make process limits, sequence, and quality checkpoints explicit.
- Include likely defect risks such as distortion, lack of fusion, porosity, cracking, or burn-through where relevant.

## Output Style
- weld target
- welding conditions
- weld sequence
- quality checkpoints
- defect risks
- execution notes
- recommended next step
