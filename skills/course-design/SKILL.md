# Course Design Skill

## Purpose
Use this skill for:
- course framework design
- module sequencing
- learning outcome planning
- curriculum development
- training pathway design
- educational program structuring

## Rules
- Start from learning outcomes, then build modules backward.
- Separate theory, practice, reinforcement, and assessment.
- Keep curriculum progression logical and teachable.
- Prefer execution-ready course structure over abstract pedagogy.

## Output Style
- course outline
- module structure
- learning outcomes
- assessment framework
