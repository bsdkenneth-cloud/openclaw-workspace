# Pipe Fitting Fabrication Skill

## Purpose
Use this skill for:
- pipe fitting planning
- spool fabrication support
- routing interpretation
- fit-up and alignment logic
- field or shop assembly preparation
- pipework execution summaries

## Core Method
Before giving pipe fitting guidance:
1. Define the actual pipework target.
2. Identify line size, material, schedule, fittings, and connection type.
3. Separate drawing intent from field-fit reality.
4. Plan cut, bevel, fit-up, alignment, tack, weld, and checking sequence.
5. Re-check whether the line can be fabricated, aligned, and installed within tolerance.

## Rules
- Start from first principles and actual pipework constraints.
- Separate routing, fitting method, weld preparation, and checking points.
- Prefer buildable pipe-fitting guidance over abstract explanation.
- Make reference points, orientation, and tolerance checks explicit.
- Include practical risks such as mismatch, offset, rotation error, pull-up stress, or installation interference where relevant.

## Output Style
- pipework target
- routing interpretation
- fitting sequence
- alignment notes
- checking points
- execution risks
- recommended next step
