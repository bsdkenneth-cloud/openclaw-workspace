# Video Script Production Skill

## Purpose
Use this skill for:
- video script drafting
- short-form content structure
- hook design
- scene breakdown
- voiceover scripting
- storyboard-ready content planning

## Rules
- Prioritize viewer retention and clarity.
- Separate hook, body, payoff, and CTA.
- Keep scripts shootable and editable.
- Prefer concise, high-impact language.

## Output Style
- concept summary
- script draft
- scene breakdown
- CTA and ending options
