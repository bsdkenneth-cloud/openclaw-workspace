# Content Automation Skill

## Purpose
Use this skill for:
- content calendar planning
- post drafting
- batch content generation
- AI content workflow design
- publishing pipeline structure
- repurposing long-form into short-form

## Rules
- Write for execution, not theory only.
- Keep outputs reusable across platforms.
- Separate draft copy, visual direction, and CTA.
- Prefer scalable workflows over one-off tasks.

## Output Style
- content brief
- draft copy
- workflow steps
- publishing recommendations
