# Data Sheet Automation Skill

## Purpose
Use this skill for:
- spreadsheet preparation
- column mapping
- row transformation
- automation-friendly data structuring
- import and export preparation
- recurring data cleanup

## Rules
- Keep field names explicit and consistent.
- Prefer automation-ready structure over manual convenience.
- Separate source data, transformed data, and output targets.
- Avoid ambiguous column naming.

## Output Style
- field mapping
- row structure
- transformation steps
- output-ready data notes
