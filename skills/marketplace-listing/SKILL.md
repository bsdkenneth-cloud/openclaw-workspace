# Marketplace Listing Skill

## Purpose
Use this skill for:
- listing title drafting
- product description writing
- marketplace positioning
- feature-to-benefit conversion
- buyer-facing listing structure
- conversion improvement for listings

## Rules
- Lead with buyer value, not only product specs.
- Keep titles searchable and descriptions readable.
- Separate core specs, selling points, and trust signals.
- Prefer practical sales wording over exaggerated hype.

## Output Style
- title options
- listing description
- key selling points
- conversion recommendations
