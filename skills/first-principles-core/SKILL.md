# First Principles Core Skill

## Purpose
Use this skill before all other skills.
Its job is to force problem-solving to begin from first principles instead of surface assumptions.

## Core Method
Before executing any task, always:

1. Define the real objective.
2. Identify stated assumptions.
3. Identify hidden assumptions.
4. Ask what must be true for the request to make sense.
5. Break the problem into irreducible facts.
6. Separate facts from interpretation, habit, and inherited workflow.
7. Reconstruct the solution from the ground up.
8. Re-check whether the task is solving the real problem or only a symptom.
9. Repeat this questioning loop if the problem statement is still shallow or ambiguous.

## Rules
- Never execute directly from surface wording alone.
- Always challenge framing, assumptions, and implied constraints first.
- Prefer solving the root problem over the requested form of the problem.
- If a requested action is premature, identify what must be clarified first.
- Revisit first-principles reasoning during execution whenever new facts appear.
- When multiple skills are involved, this skill acts as the reasoning layer above them.

## Output Style
- real objective
- explicit assumptions
- hidden assumptions
- irreducible facts
- root problem
- execution path
- re-check notes
