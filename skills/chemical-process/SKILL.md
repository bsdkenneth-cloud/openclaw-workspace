# Chemical Process Skill

## Purpose
Use this skill for:
- chemical process analysis
- treatment workflow design
- material and reagent review
- process troubleshooting
- industrial chemistry support
- stepwise treatment planning

## Rules
- Start from the actual process goal and chemistry involved.
- Separate feedstock, treatment method, reaction logic, and downstream handling.
- Prefer practical process clarity over vague chemical claims.
- Make hazards, limits, and uncertainties explicit.

## Output Style
- process summary
- treatment logic
- material notes
- operational cautions
