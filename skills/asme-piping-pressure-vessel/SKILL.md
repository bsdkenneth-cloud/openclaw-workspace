# ASME Piping Pressure Vessel Skill

## Purpose
Use this skill for:
- ASME code-context review
- piping fabrication support
- pressure vessel fabrication context
- welding and fit-up awareness
- inspection and execution summaries
- practical code-aware shop or field guidance

## Core Method
Before giving ASME-related guidance:
1. Define the actual piping or vessel scope.
2. Identify component type, service context, material, and construction stage.
3. Separate code intent from shop or field execution limits.
4. Identify relevant fabrication, fit-up, welding, examination, and inspection considerations.
5. Re-check whether the guidance matches the real piping or vessel construction context.

## Rules
- Start from first principles and actual fabrication or construction conditions.
- Separate code context, execution logic, and inspection awareness.
- Prefer practical piping and vessel guidance over vague code references.
- Make assumptions and unknowns explicit.
- Include caution where code section, project specification, or design basis may control the answer.
- Do not pretend compliance is confirmed when project details are incomplete.

## Output Style
- scope definition
- code context
- fabrication or construction notes
- welding or fit-up notes
- inspection awareness
- cautions and unknowns
- recommended next step
