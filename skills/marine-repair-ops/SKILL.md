# Marine Repair Ops Skill

## Purpose
Use this skill for:
- ship repair planning
- maintenance scope definition
- yard coordination
- repair sequence planning
- vessel-side execution support
- operational repair summaries

## Rules
- Keep repair logic practical and sequence-aware.
- Separate defect, cause, repair scope, and execution risk.
- Avoid vague maintenance wording.
- Prefer actionable shipyard-style communication.

## Output Style
- repair scope summary
- execution notes
- risk points
- maintenance recommendations
