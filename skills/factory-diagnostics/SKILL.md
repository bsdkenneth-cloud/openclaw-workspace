# Factory Diagnostics Skill

## Purpose
Use this skill for:
- factory problem diagnosis
- process bottleneck analysis
- equipment issue structuring
- production troubleshooting
- failure-point review
- operational improvement support

## Rules
- Start from the actual production problem, not surface symptoms only.
- Separate symptom, probable cause, operational impact, and corrective action.
- Prefer practical troubleshooting logic over abstract theory.
- Make constraints and risks explicit.

## Output Style
- diagnosis summary
- probable causes
- impact notes
- corrective actions
