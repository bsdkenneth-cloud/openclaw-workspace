# Fitter Layout Skill

## Purpose
Use this skill for:
- fitter work planning
- layout marking
- alignment checking
- reference-point control
- fit-up preparation
- assembly support summaries

## Core Method
Before giving fitter or layout guidance:
1. Define the actual assembly target.
2. Identify datum lines, centerlines, elevations, and reference points.
3. Separate drawing dimensions from field-fit realities.
4. Plan marking, checking, aligning, fitting, and re-checking sequence.
5. Re-check whether the assembly can be aligned, held, and measured correctly.

## Rules
- Start from first principles and real fit-up conditions.
- Prefer measurable layout logic over vague fitting language.
- Separate reference points, temporary fit-up method, and final alignment method.
- Make tolerances and checking points explicit.
- Include likely mismatch or distortion risks where relevant.

## Output Style
- assembly target
- reference points
- layout sequence
- fit-up notes
- checking points
- execution risks
- recommended next step
