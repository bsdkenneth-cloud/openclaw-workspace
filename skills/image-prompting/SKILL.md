# Image Prompting Skill

## Purpose
Use this skill for:
- image prompt drafting
- composition planning
- style direction
- subject description refinement
- visual consistency guidance
- AI image generation preparation

## Rules
- Be explicit about subject, style, camera angle, lighting, and background.
- Separate mandatory elements from optional enhancements.
- Prefer visually concrete wording over vague adjectives.
- Keep prompts generation-ready.

## Output Style
- image prompt
- negative prompt suggestions
- visual direction notes
- variation options
