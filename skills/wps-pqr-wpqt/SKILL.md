# WPS PQR WPQT Skill

## Purpose
Use this skill for:
- WPS drafting support
- PQR interpretation
- WPQT or welder qualification logic
- essential variable review
- welding parameter control
- code-aware qualification summaries

## Core Method
Before giving WPS, PQR, or qualification guidance:
1. Define the actual welding scope and code context.
2. Identify base material, thickness range, process, position, filler, and joint type.
3. Separate procedure requirements from production convenience.
4. Identify which variables are essential, supplementary essential, or nonessential where relevant.
5. Re-check whether the procedure or qualification logic actually supports the intended production welds.

## Rules
- Start from first principles and actual welding qualification intent.
- Separate WPS, PQR, and welder qualification purposes clearly.
- Prefer qualification logic that matches real production conditions.
- Make variable ranges, limits, and qualification scope explicit.
- Include caution where code interpretation, transferability, or production fit may be uncertain.
- Prefer code-aware practical guidance over vague welding commentary.

## Output Style
- welding scope
- qualification context
- variable review
- WPS or PQR notes
- qualification logic
- limits and cautions
- recommended next step
