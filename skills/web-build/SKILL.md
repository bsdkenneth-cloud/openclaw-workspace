# Web Build Skill

## Purpose
Use this skill for:
- website planning
- landing page structure
- page section design
- UI copy drafting
- conversion-oriented web messaging
- implementation-ready website logic

## Rules
- Prefer clean structure over decoration.
- Separate layout logic, copy, and CTA.
- Keep outputs implementation-friendly.
- Focus on clarity, credibility, and conversion.

## Output Style
- sitemap
- page outline
- section copy
- CTA recommendations
